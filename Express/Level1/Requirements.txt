CodeSchool Express Level 1

Create a simple express node app meeting the following requirements:

1. Create an express app.

2. Create a root route that returns “Hello World”

4. Create a ‘/name’ route that returns your name

5. Create a /redirect route that sends you to 
/surprise with a moved permanently status code

6. Create a route that returns the current date. 
You will need to look up how to get the current date.
